# Node Console Toolkit

App 1 del Backend Node Expert Lab.

CLI sencilla creada con Node.js y TypeScript para practicar fundamentos backend sin framework.

## Objetivo

Practicar conceptos importantes de Node:

- process.argv
- fs
- path
- crypto
- lectura y escritura de JSON
- manejo de errores en consola
- scripts reutilizables

## Stack

- Node.js 20.19.4
- TypeScript 5.9.3

## Scripts

```bash
npm run dev
npm run build
npm run start
Ejecutar en desarrollo
npm run dev
Ejecutar con argumentos
